* server目录
  * app 源码
    * servers 各类服务器
    * util 工具类
  * config 配置项
  * logs 日志打印存放目录
  * app.js 程序入口文件
  * package.json 包依赖文件

使用方式：
  1. git clone https://github.com/zhenghan2017/2018.git
  2. cd server
  3. npm install
  4. pomelo start
